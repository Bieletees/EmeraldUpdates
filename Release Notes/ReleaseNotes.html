Version 0.1.2

In version 0.1.2, Emerald for Mac is now more stable and filled with new features

Here's what's new:

  • There's now a translucent window for a better aesthetic!
  • Favorites are now located at the top and the Table selector at the bottom
  • JavaScript alerts are now available
  • Added new Keyboard Shortcuts such as Command + L
  • Reduced animations on the sidebar to be more efficient
  • Added symbols to the Traffic Lights
  • The sidebar is now made from pure Liquid Glass in macOS Tahoe to follow other macOS Applications
  • LOTS of bug fixes

Thanks for using Emerald for Mac!

Version 0.1.1

In version 0.1.1, Emerald for Mac has new features and it's smoother

Here's what's new:

  • Emerald now features a Command Bar instead of New Tabs
  • Added some credits in the About Emerald window
  • You can now close Tabs pressing Command + Shift + W
  • Page Zoom is now available
  • You can now start searching when you open the app
  • LOTS of bug fixes

Thanks for using Emerald for Mac!
