Version 0.2.0

In version 0.2.0, Emerald for Mac enters public Alpha and has lots of requested features.

Here's what's new:

  • Emerald for Mac is now a public Alpha. Starting today, if someone wants to try Emerald for Mac, they can do it right away without touching any waitlist!
  • Introducing Profiles! With Profiles, you can makes Tables truly separate. Each Profile has its own data.
  • You can now reorder Pinned Tabs, Folders and Tabbies
  • Fixed corner radius and traffic light colors on macOS Sonoma and Sequoia
  • Added the Name section, so you can have a better experience with Emerald
  • To create New Tabbies, there's a new button in the Tabbies menu or in the bottom of the screen. The plus button turns into a menu where you can add new Tabbies, Tables and Folders
  • LOTS of bug fixes

Thanks for using Emerald for Mac!

Version 0.1.4

In version 0.1.4, Emerald for Mac has new features, and tons of bugs have been fixed.

Here's what's new:

  • Introducing Site Search! With Site Search, you can search on a website, just like you do with a Search Engine! There are some websites included, but you can add as many as you want.
  • Find on Page is now fully working!
  • Changed the lock icon for the Tab icon
  • The ad blocker is now stronger
  • The new File menu changed the Shortcuts to Command + W (Close Tab) and Shift + Command + W (Close Window)
  • Reduced Sidebar closing and opening animations duration for a faster experience
  • Website overlays (Find on Page, download indicators...) are now made of Liquid Glass (macOS Tahoe) and Thin Material (Sequoia and Sonoma)
  • LOTS of bug fixes

Other features from previous versions:

  • Each Table has its own Tabbies section
  • When switching Tables, the active Tab changes to the current Table

Thanks for using Emerald for Mac!

Version 0.1.3

In version 0.1.3, Emerald for Mac has some new features thanks to your feedback, and lots of bugs have been fixed, most of them happened on macOS Sonoma and Sequoia.

Here's what's new:

  • Fixed color scheme for window tinting
  • There are more Sidebar settings. You can now disable top Favorites and enable Tree Tabs, a new feature from this update!
  • Introducing Tree Tabs! With Tree Tabs, every Tab opened in a new Tab, it will show as a sub tab from that Tab
  • Fixed some bugs that produced chrashes
  • Fixed LOTS of issues that were present on macOS Sonoma and Sequoia
  • Tabs view from Tables in Feed now follows all the new features
  • LOTS of bug fixes

Thanks for using Emerald for Mac!
